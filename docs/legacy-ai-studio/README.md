# Legacy AI Studio Scaffolding

This directory preserves files inherited from the original Google AI Studio
project template that generated this Android app. They are **not** consumed
by Gradle, the Android build, or any source code in this repository.

| File              | Origin                              | Status                      |
|-------------------|-------------------------------------|-----------------------------|
| `metadata.json`   | AI Studio project descriptor        | Unused by build, kept for archive |

If you need to re-import the project into AI Studio, these files are the
minimum required for the studio to recognize the original prompt context.
